# JavaScript DOM & localStorage Toolkit

## 📌 Project Overview
This project is a beginner-friendly To-Do List web application built using JavaScript DOM manipulation and localStorage.

It demonstrates how to:
- dynamically update a webpage using JavaScript
- store data in the browser
- persist data after page refresh

---

## 🎯 Features
- Add tasks
- Edit tasks
- Delete tasks
- Mark tasks as complete
- Persistent storage using localStorage

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript (DOM Manipulation)
- localStorage

---

## ⚙️ How to Run the Project

1. Download or clone the project folder
2. Open the folder in VS Code
3. Open `index.html` in your browser

OR

Use Live Server in VS Code:
- Right click `index.html`
- Click **Open with Live Server**

## 🧠 How It Works

- User inputs a task
- JavaScript creates a new list item using DOM manipulation
- Task is saved in localStorage
- On page reload, tasks are retrieved and displayed

---

## ⚠️ Known Issues
- Tasks are stored locally and not synced across devices
- No backend database is used
